## 搭建 ec-cli 脚手架

### 安装

```shell
# use npm
npm install -g ec-cli

# or yarn
yarn global add ec-cli
```

### 使用

#### 快速开始

```shell
ec create <name> [-f|--force]
```

#### 选项

-`-f,--force`: 如果项目存在则重写
